ISweep Local Test

Quick steps

  1. Place this file, captions.vtt, and (optional) test.mp4 in the same folder.
  2. From this folder run: python -m http.server 8080
  3. Open: http://127.0.0.1:8080/isweep_test.html
  4. If you must use file URLs, enable "Allow access to file URLs" for the ISweep extension.
  5. Backend check: http://127.0.0.1:8001/docs
